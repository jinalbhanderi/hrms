Create Break

1. You can not delete the break Policy if the shift is assigned to the Break Policy.

2. Start break End break :- Here you can pass break end time more then break start time which will be considered as two day concept.

Eg. Break Start time 11: 00 PM current day break end time 12 : 00 AM for very next day as its 12 AM the total break hour is one hour.

3. If the fixed break time is more then break end time then system will add the extra time passed in the fixed break duration in break end time.

Eg. Break start time is 10 AM, Break end time is 11 AM and fixed break duration is 2 Hour. So in this case break end time will be automatically updated.
